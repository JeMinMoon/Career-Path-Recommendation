# need to setup
## PyPDF2
pdf를 text로 바꿀 수 있는 python 라이브러리이다.  
### install
```
pip install pypdf2
```
## OpenAI Python library
openai api를 사용하기 위한 라이브러리이다.
### install
```
pip install --upgrade openai
```
### 환경변수 설정
https://platform.openai.com/docs/quickstart  
내PC 우클릭-> 속성-> 고급 시스템 설정-> 환경변수-> 시스템 변수 새로 만들기  
변수명: OPENAI_API_KEY  
변수값: openai에서 발급받은 키
